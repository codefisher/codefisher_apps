Pastel SVG Icons

About

Pastel SVG is an icon set based on the popular silk icons found at FamFamFam.com. It uses the same style, and the icons are virtually identical. The goal will to be to soon have all the icons that are contained in the original set.

The greatest addition of this set, is that the icons where created as Sclable Vector Graphics which enables the icons to be created at many larger sizes then only the 16x16 that existed in the original set.

Getting the SVG source files

All the of the above downloads only provide the PNG files that are generated from the SVG files that I created. If you would like to use the SVG files, I ask that you make a donation. Please see the donation page for more details. Once you have made the donation you will be able to download them from the Pastel SVG Downloads page.

What is Next

Currently, though the Pastel SVG Icon set has more icons then the original, it still last 250+ icons from the silk icon set. We are working on adding them soon.

Also, many of the 24x24 icons don't look their best. This is because they are scaled 1.5x from the 16 icons, which causes many of the lines not to align perfectly causing the icons to look slightly blurred. To fix this it is necessary to created new version of the icons that have been retouched for this size. Currently this has been done for 100+ icons, but that still leaves a long way to go.

Finally we are very open to requests for new icons that might be a valuable addition to the icon set, please see our Icon Request Page.
